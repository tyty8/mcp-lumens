## Descripción general

Braintree Pagos MCP conecta tu asistente de IA con Braintree, la pasarela de pagos de PayPal especialmente valorada por empresas que necesitan aceptar tarjetas de crédito y débito, PayPal, Venmo y carteras digitales como Apple Pay y Google Pay en un único sistema integrado. Al ser propiedad de PayPal y contar con su infraestructura global de seguridad, Braintree es una opción de confianza para negocios que priorizan la seguridad y la cobertura de métodos de pago por encima de todo.

El servidor permite al asistente de IA realizar operaciones de pago completas: procesar cobros con tarjeta, gestionar el vault de clientes (donde se almacenan de forma segura los datos de pago para cobros recurrentes), crear y gestionar suscripciones, emitir reembolsos y consultar el historial de transacciones. Para negocios que han optado por Braintree como su procesador de pagos principal, este servidor transforma la gestión de pagos en una tarea conversacional.

La gestión del vault de clientes es especialmente valiosa para negocios con clientes recurrentes: los datos de tarjeta se guardan de forma segura en la infraestructura de Braintree (nunca en los servidores del negocio), y el asistente puede realizar cobros futuros a un cliente guardado sin que este tenga que volver a introducir sus datos de pago. Esto es fundamental para suscripciones mensuales, abonos de temporada, membresías de gimnasio o cualquier modelo de facturación periódica.

## Casos de uso para negocios

- **Gimnasios y centros deportivos:** Cobro automático de la cuota mensual a todos los socios con tarjeta registrada en el vault, con gestión de los intentos fallidos y notificación automática cuando un pago no se puede procesar.
- **Academias y centros de formación:** Facturación de mensualidades de cursos, con gestión de descuentos por familia numerosa, promociones de matrícula y cobro de materiales adicionales a los clientes guardados.
- **Clínicas de tratamientos estéticos y medicina:** Cobro de paquetes de sesiones prepagadas, gestión de abonos de tratamiento y procesamiento de reembolsos parciales cuando un cliente cancela parte de un paquete.
- **Servicios de suscripción y membresía:** Gestión del ciclo completo de suscripción — alta, cobro mensual, actualizaciones de plan, pausas temporales y bajas — con toda la lógica de pago gestionada por el asistente.
- **Agencias de alquiler a largo plazo:** Cobro mensual de rentas a inquilinos con tarjeta registrada, con gestión automática de los intentos de cobro fallidos y comunicación con el inquilino en caso de problema.
- **Consultoras con clientes recurrentes:** Cobro mensual de fees de retainer a clientes con contratos de prestación continuada de servicios, sin necesidad de emitir una factura manual y esperar el pago cada mes.

## Configuración básica

El servidor requiere credenciales de una cuenta de Braintree:

```bash
git clone https://github.com/QuentinCody/braintree-mcp-server
cd braintree-mcp-server
npm install
npm run build
```

Configura el servidor con tus credenciales de Braintree:

```json
{
  "mcpServers": {
    "braintree": {
      "command": "node",
      "args": ["dist/index.js"],
      "env": {
        "BRAINTREE_ENVIRONMENT": "production",
        "BRAINTREE_MERCHANT_ID": "tu-merchant-id",
        "BRAINTREE_PUBLIC_KEY": "tu-public-key",
        "BRAINTREE_PRIVATE_KEY": "tu-private-key"
      }
    }
  }
}
```

Accede al panel de Braintree (braintreegateway.com) para obtener tu Merchant ID y las claves pública y privada en la sección Account > My User > API Keys. Para pruebas, cambia `BRAINTREE_ENVIRONMENT` a `sandbox` y usa las credenciales del entorno de pruebas, que te permiten procesar pagos simulados sin cargo real.
