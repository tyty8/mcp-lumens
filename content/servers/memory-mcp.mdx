## Descripción general

El Servidor MCP de Memoria resuelve una de las limitaciones fundamentales de los modelos de lenguaje: la ventana de contexto se restablece entre conversaciones. Al mantener un grafo de conocimiento persistente almacenado en disco, este servidor permite que un cliente de IA recuerde hechos, relaciones y observaciones entre sesiones. Las entidades (personas, proyectos, conceptos) y las relaciones entre ellas se almacenan como un grafo estructurado y se recuperan bajo demanda.

Este servidor es especialmente valioso para construir asistentes de IA que acumulan conocimiento con el tiempo: recordar sus preferencias, rastrear el estado de proyectos de larga duración o mantener una base de conocimiento compartida para un equipo. El grafo de conocimiento se almacena como un archivo JSON en disco, lo que facilita su inspección, copia de seguridad y control de versiones.

## Configuración

De forma predeterminada, el grafo de conocimiento se almacena en un archivo `memory.json` en el directorio de trabajo del servidor. Puede personalizar la ruta de almacenamiento con la variable de entorno `MEMORY_FILE_PATH`.

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"],
      "env": {
        "MEMORY_FILE_PATH": "/Users/yourname/.claude/memory.json"
      }
    }
  }
}
```

## Capacidades principales

- Crear, actualizar y eliminar entidades nombradas con atributos tipados
- Almacenar relaciones entre entidades (p. ej., "Alice trabaja-en Acme Corp")
- Agregar y eliminar observaciones (hechos con marca de tiempo) en cualquier entidad
- Buscar entidades por nombre o tipo
- Recuperar el grafo completo o subgrafos alrededor de una entidad específica
- El almacenamiento persistente sobrevive a reinicios del proceso y nuevas sesiones de conversación
