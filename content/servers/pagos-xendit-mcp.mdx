## Descripción general

Pagos Xendit MCP integra tu asistente de IA con Xendit, la pasarela de pagos líder en el Sudeste Asiático que opera en Indonesia, Filipinas, Tailandia, Vietnam y Malasia. Para empresas que tienen operaciones, clientes o proveedores en esta región, Xendit es el equivalente local de Stripe o PayPal: una infraestructura de pagos robusta y confiable adaptada a los métodos de pago locales predominantes en cada país, incluyendo transferencias bancarias locales, wallets digitales regionales y pagos en efectivo en puntos de conveniencia.

El servidor permite al asistente de IA gestionar todo el ciclo de pagos: crear facturas de pago (payment links) para cobrar a clientes, realizar desembolsos a proveedores o socios, consultar el saldo disponible en la cuenta de Xendit y ejecutar transferencias bancarias. Para empresas con equipos o proveedores distribuidos por el Sudeste Asiático, la capacidad de gestionar pagos en múltiples países y monedas desde un asistente de IA en español supone una simplificación operativa enorme.

La particularidad del Sudeste Asiático en cuanto a métodos de pago hace que una pasarela local como Xendit sea indispensable: en Indonesia, el pago mediante QRIS (código QR interbancario) y los wallets como GoPay y OVO son dominantes; en Filipinas, GCash y Maya son ubicuos; en Tailandia, PromptPay es el estándar de transferencias instantáneas. Xendit abstrae toda esta complejidad, y este servidor MCP la lleva al asistente de IA.

## Casos de uso para negocios

- **Empresas exportadoras a Sudeste Asiático:** Cobro de facturas a distribuidores y clientes en Indonesia o Filipinas mediante métodos de pago locales, sin necesidad de establecer entidades bancarias locales.
- **Plataformas de comercio electrónico con presencia en ASEAN:** Gestión automatizada de los cobros a compradores y los pagos a vendedores locales en los países donde opera la plataforma.
- **Empresas de servicios digitales internacionales:** Facturación y cobro a clientes corporativos en Tailandia, Vietnam o Malasia mediante transferencias bancarias locales que resultan más económicas que los pagos internacionales con tarjeta.
- **Agencias de viajes con operadores en Sudeste Asiático:** Pago a operadores turísticos, guías y alojamientos locales mediante desembolsos en moneda local a través de Xendit, evitando comisiones de cambio de divisa elevadas.
- **Empresas de BPO y outsourcing:** Pago de nóminas o servicios a equipos y proveedores en los países del Sudeste Asiático cubiertos por Xendit, con transferencias bancarias locales más rápidas y económicas.
- **Comercio mayorista con distribuidores asiáticos:** Gestión de cobros de pedidos grandes mediante facturas de pago (invoices) de Xendit con condiciones de pago negociadas y seguimiento automático del estado.

## Configuración básica

El servidor requiere una cuenta de Xendit y tu API key de producción o sandbox:

```bash
git clone https://github.com/mrslbt/xendit-mcp
cd xendit-mcp
npm install
npm run build
```

Configura el servidor con tus credenciales de Xendit:

```json
{
  "mcpServers": {
    "xendit-pagos": {
      "command": "node",
      "args": ["dist/index.js"],
      "env": {
        "XENDIT_SECRET_KEY": "xnd_production_xxxxxxxxxxxxxxxx",
        "XENDIT_WEBHOOK_TOKEN": "tu-webhook-verification-token",
        "XENDIT_ENVIRONMENT": "production"
      }
    }
  }
}
```

Regístrate en xendit.com para crear tu cuenta. Xendit requiere un proceso de verificación KYB (Know Your Business) para las cuentas de producción. Puedes comenzar con el entorno sandbox (usando la clave `xnd_development_...`) para probar la integración antes de activar pagos reales. La documentación completa de la API está disponible en docs.xendit.co.
