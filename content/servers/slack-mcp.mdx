## Descripción general

El Servidor MCP de Slack integra su cliente de IA con espacios de trabajo de Slack, permitiéndole participar directamente en flujos de trabajo de comunicación del equipo. En lugar de copiar y pegar mensajes hacia y desde una ventana de chat de IA, el servidor permite que la IA lea el contexto de la conversación, redacte y envíe mensajes, y busque en el historial de su espacio de trabajo sin salir de la conversación.

Los casos de uso típicos incluyen resumir un hilo largo de Slack, redactar un mensaje de anuncio para revisión antes de publicarlo, monitorear un canal en busca de palabras clave específicas, o automatizar notificaciones rutinarias desde scripts y pipelines de CI.

## Configuración

Necesita un Bot Token de Slack (`xoxb-...`) de una aplicación de Slack configurada con los alcances OAuth apropiados. Como mínimo, el bot requiere `channels:read`, `channels:history`, `chat:write` y `users:read`. Establezca el token como la variable de entorno `SLACK_BOT_TOKEN`.

```json
{
  "mcpServers": {
    "slack": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-slack"],
      "env": {
        "SLACK_BOT_TOKEN": "xoxb-yourTokenHere"
      }
    }
  }
}
```

El bot debe ser invitado a cualquier canal privado al que necesite acceder con `/invite @nombre-de-su-bot`.

## Capacidades principales

- Listar canales públicos y sus metadatos
- Leer el historial de mensajes de los canales a los que el bot se ha unido
- Publicar mensajes en canales o como respuestas en hilos
- Listar miembros del espacio de trabajo y su información de perfil
- Buscar mensajes en todo el espacio de trabajo
- Reaccionar a mensajes con emoji
