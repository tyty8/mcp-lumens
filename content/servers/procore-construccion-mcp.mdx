## Descripción general

El servidor Procore Gestión de Construcción MCP proporciona acceso conversacional a la plataforma líder de gestión de proyectos de construcción. Procore es el software de referencia para grandes constructoras, promotoras y empresas de ingeniería civil, y este servidor permite que los equipos de obra interactúen con todos sus datos —proyectos, documentación técnica, presupuestos, incidencias y comunicaciones— mediante lenguaje natural, sin necesidad de navegar constantemente por la interfaz web de Procore.

La gestión de RFIs (Requests for Information) es uno de los flujos de trabajo más críticos en cualquier proyecto de construcción. Cuando un contratista o subcontratista necesita aclaración sobre un plano, una especificación técnica o un procedimiento, genera una RFI que debe ser respondida por el arquitecto o el director de obra en tiempo y forma. Un retraso en la respuesta a una RFI puede paralizar los trabajos en un frente de obra, con el consiguiente coste en horas de trabajadores y maquinaria inactivos. El asistente con este servidor puede monitorear el estado de todas las RFIs abiertas, alertar sobre las que llevan más días sin respuesta y ayudar a priorizar según el impacto en la planificación de obra.

Los registros diarios de campo son otro elemento fundamental de la gestión documental en construcción. Registrar cada día las condiciones meteorológicas, el personal presente, los trabajos ejecutados, los materiales recibidos y las incidencias ocurridas es una obligación contractual y legal que protege a la empresa en caso de reclamaciones o disputas. El asistente puede ayudar a generar y completar estos registros de forma eficiente durante o al final de cada jornada, garantizando que la documentación de obra está siempre al día.

## Casos de uso para negocios

- **Dirección de obra y project managers:** Consulta en tiempo real del estado de todas las RFIs, submittals y punch list items de un proyecto sin necesidad de abrir Procore, con resúmenes diarios de los puntos críticos que requieren atención.
- **Constructoras con múltiples proyectos simultáneos:** Visión consolidada del estado de todos los proyectos activos —avance, alertas presupuestarias y cuellos de botella documentales— en una sola consulta conversacional.
- **Departamentos de compras y presupuestos:** Análisis del presupuesto ejecutado vs. presupuesto previsto con detección de partidas en riesgo de desviación, para actuar preventivamente antes de que los sobrecostes sean inevitables.
- **Subcontratistas y proveedores:** Acceso rápido al estado de sus submittals de materiales y equipos, verificando si han sido aprobados por el director de obra o si están pendientes de revisión con observaciones.
- **Equipos de seguridad en obra:** Registro y seguimiento de incidencias de seguridad, inspecciones y observaciones de los técnicos de prevención, asegurando que cada punto se documenta correctamente y se escala si es necesario.
- **Administración y facturación:** Consulta del estado de certificaciones de obra y validación de los trabajos ejecutados para emitir facturas alineadas con el avance real del proyecto certificado por la dirección facultativa.

## Configuración básica

Registra tu empresa en el programa de desarrolladores de Procore y crea una aplicación para obtener las credenciales OAuth:

```bash
npm install -g @tylerilunga/procore-mcp-server
```

Configura las credenciales en tu cliente MCP:

```json
{
  "mcpServers": {
    "procore": {
      "command": "procore-mcp-server",
      "env": {
        "PROCORE_CLIENT_ID": "tu-client-id",
        "PROCORE_CLIENT_SECRET": "tu-client-secret",
        "PROCORE_REDIRECT_URI": "http://localhost:3000/callback",
        "PROCORE_COMPANY_ID": "tu-company-id"
      }
    }
  }
}
```

La autenticación de Procore usa OAuth 2.0. En el primer uso, el servidor abrirá un navegador para que autorices el acceso a tu cuenta de Procore. El company ID se encuentra en la URL de tu panel de Procore. Para acceder a proyectos específicos, asegúrate de que el usuario tiene los permisos necesarios en Procore para cada módulo que quieras usar.
