## Descripción general

El Servidor MCP de SQLite hace que sea muy sencillo dar a un cliente de IA acceso a una base de datos SQLite local. Sin procesos de servidor, sin gestión de cadenas de conexión: simplemente apunte el servidor a un archivo `.sqlite` y su IA podrá inmediatamente consultar, insertar, actualizar e inspeccionar los datos que contiene.

Este servidor es muy adecuado para bases de datos de desarrollo local, aplicaciones prototipo y flujos de trabajo de análisis de datos sin conexión. Dado que SQLite está integrado y basado en archivos, la configuración es mínima y la base de datos viaja con su proyecto. La IA puede ayudarle a explorar el esquema, escribir consultas, inicializar datos de prueba y depurar problemas de datos, todo en una sola conversación.

## Configuración

Pase la ruta a su archivo SQLite como primer argumento. La ruta puede ser relativa o absoluta. Si el archivo no existe, SQLite lo creará.

```json
{
  "mcpServers": {
    "sqlite": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-sqlite",
        "/Users/yourname/projects/app/local.db"
      ]
    }
  }
}
```

## Capacidades principales

- Ejecutar consultas SQL arbitrarias (SELECT, INSERT, UPDATE, DELETE)
- Listar todas las tablas y vistas de la base de datos
- Inspeccionar definiciones de columnas, tipos y restricciones de cualquier tabla
- Crear y modificar tablas mediante instrucciones DDL
- No se requiere servidor de base de datos externo ni conexión de red
- Las lecturas y escrituras persisten en el archivo entre sesiones
