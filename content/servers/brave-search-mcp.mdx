## Descripción general

El Servidor MCP de Brave Search conecta su cliente de IA al índice de búsqueda independiente de Brave, dándole acceso a contenido web actualizado sin depender únicamente de los datos de entrenamiento. Esto es particularmente útil para preguntas sobre eventos recientes, documentación actual, precios o cualquier información que cambie con frecuencia.

Brave Search está orientado a la privacidad y opera su propio rastreador web, lo que significa que los resultados no están filtrados a través del mismo prisma que otros motores de búsqueda principales. El servidor admite tanto búsqueda web general como búsqueda de negocios y lugares locales, lo que lo hace útil también para consultas con conciencia de ubicación.

## Configuración

Se requiere una clave de API de Brave Search. Regístrese en [brave.com/search/api](https://brave.com/search/api) para obtener una. Establezca la clave como la variable de entorno `BRAVE_API_KEY`.

```json
{
  "mcpServers": {
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-brave-search"],
      "env": {
        "BRAVE_API_KEY": "BSAyourKeyHere"
      }
    }
  }
}
```

El nivel gratuito de la API de Brave Search incluye un número generoso de consultas mensuales, lo que es suficiente para uso personal y desarrollo ligero.

## Capacidades principales

- Búsqueda web general que devuelve títulos, URLs y fragmentos de resultados
- Búsqueda de negocios y lugares locales con direcciones y valoraciones
- Número de resultados configurable por consulta
- Índice respetuoso con la privacidad, independiente de los principales proveedores de búsqueda
- Sin seguimiento de usuarios ni elaboración de perfiles por parte de Brave
