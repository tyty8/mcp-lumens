## Descripción general

El Servidor MCP Everything no es una herramienta de producción: es una implementación de referencia completa diseñada para ejercitar cada capacidad de la especificación MCP. Si está construyendo un cliente MCP o evaluando una aplicación host, este servidor le permite verificar que las herramientas, recursos, prompts y el muestreo funcionan correctamente sin necesidad de configurar servicios externos reales.

El servidor incluye una amplia variedad de herramientas de ejemplo (echo, aritmética, recursos similares a archivos, operaciones lentas para probar tiempos de espera), tipos de recursos diversos (texto, binario, URIs con plantillas) y prompts prediseñados. Ejecutar este servidor contra su cliente le brinda visibilidad inmediata sobre qué funcionalidades de MCP son compatibles y cómo se gestionan los casos límite.

## Configuración

No se requieren claves de API ni dependencias externas. Inicie el servidor y apunte su cliente MCP hacia él.

```json
{
  "mcpServers": {
    "everything": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-everything"]
    }
  }
}
```

## Capacidades principales

- Herramientas de ejemplo que cubren operaciones síncronas, asíncronas y de larga duración
- Múltiples tipos de recursos: texto estático, datos binarios y recursos con plantillas URI
- Plantillas de prompts prediseñadas que demuestran la funcionalidad de prompts de MCP
- Soporte de muestreo para probar la invocación de LLM en el lado del cliente
- Casos de error intencionales para probar el manejo de errores del cliente
- Útil como prueba de humo al actualizar bibliotecas de cliente MCP
