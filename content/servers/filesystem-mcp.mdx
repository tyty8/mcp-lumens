## Descripción general

El Servidor MCP de Sistema de Archivos expone su sistema de archivos local a los clientes de IA de forma controlada y auditable. Usted especifica uno o más directorios raíz al iniciar el servidor, y este restringe todas las operaciones a esas rutas: la IA no puede leer ni escribir fuera de los límites que usted defina. Esto lo hace práctico para asistencia de IA a nivel de proyecto sin exponer todo el disco.

Entre los casos de uso más comunes se incluyen permitir que una IA lea su código base para responder preguntas, escribir nuevos archivos, refactorizar código existente en múltiples archivos o generar archivos de configuración en las ubicaciones correctas. Combinado con un cliente de IA capaz, este servidor le proporciona efectivamente un programador en pareja con IA y acceso directo a los archivos.

## Configuración

Pase una o más rutas de directorio permitidas como argumentos al lanzar el servidor. Todas las rutas deben ser absolutas.

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/yourname/projects/my-app",
        "/Users/yourname/documents"
      ]
    }
  }
}
```

## Capacidades principales

- Leer contenido de archivos (texto y binario)
- Escribir y sobrescribir archivos
- Listar contenido de directorios de forma recursiva
- Crear y eliminar directorios
- Mover y renombrar archivos
- Buscar contenido de archivos con coincidencia de patrones
- Todas las operaciones están aisladas en las rutas raíz configuradas
